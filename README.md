# 😷 Face Mask Detection using Deep Learning

## 📌 About the Project
This project is about detecting whether a person is wearing a face mask or not using deep learning. I built this project to understand how computer vision works in real-time applications.

## 🚀 What this project does
- Detects faces with mask and without mask  
- Works with images and can also be used for real-time detection  
- Uses computer vision techniques for analysis  

## 🛠 Tools & Technologies
- Python  
- OpenCV  
- TensorFlow / Keras  
- NumPy  

## 📊 What I learned
- Basics of image processing  
- How to train a deep learning model  
- Working with real-time detection using webcam  

## ▶️ How to run
1. Install required libraries  
2. Open the notebook file (.ipynb)  
3. Run all the cells  
4. Start webcam for real-time detection
